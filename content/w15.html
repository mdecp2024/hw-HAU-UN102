題目一:從 1 累加到 100

# 初始化變數
sum = 0
init = 1
upto = 100

# 使用 for 迴圈進行累加
for i in range(init, upto + 1):
sum += i # sum = sum + i 的簡寫

# 顯示結果
print(f"從 {init} 累加到 {upto} 的總和是: {sum}")

題目四:add_avoid_8

# 定義函式 add_avoid_8
def add_avoid_8(start, end):
# 初始化總和變數
total = 0

# 使用 for 迴圈遍歷範圍
for i in range(start, end + 1):
# 檢查數字是否包含 "8"
if '8' not in str(i): # 如果數字轉為字串後不包含 "8"
total += i # 累加

# 返回總和
return total

# 呼叫函數並顯示結果
result = add_avoid_8(1, 100)
print(f"從 1 累加到 100 (避開包含 '8' 的數字) 的總和是: {result}")

心得:

程式設計過程中，我學會了如何使用 Python 進行數字範圍的條件檢查和累加計算。我體會到將邏輯變成函式的重要性，這不僅讓程式碼更具可塑性，也提高了程式的可讀性。每個題目的設計思考和程式實現過程都讓我加深了對 Python 基本語法結構的理解，尤其是在條件判斷、字串操作和迴圈結構方面的應用。

避開8和3的整數

def add_avoid_8(init, addto):
total = 0 # 初始化總和
for i in range(init, addto + 1): # 遍歷從 init 到 addto 的數字
# 如果數字轉換為字串後包含 "8" 或 "3"，則跳過這個數字
if '8' in str(i) or '3' in str(i):
continue
total += i # 累加不包含 "8" 或 "3" 的數字
return total # 返回累加的總和

# 呼叫函式並打印結果
result = add_avoid_8(1, 100)
print(f"從 1 累加到 100（避開包含 '8' 或 '3' 的數字）的總和是: {result}")
